#  Tip Calculator App

A simple and intuitive **React Tip Calculator** that helps users calculate the total bill by averaging service ratings from two people. The app updates instantly as inputs change and provides a clean, user-friendly experience.

---

##  Features

* Enter total bill amount
* Rate service quality for yourself and a friend
* Automatic average tip calculation
* Real-time total amount display
* Reset all inputs with one click
* Clean, responsive UI
